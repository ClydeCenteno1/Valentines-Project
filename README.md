A fully responsive, interactive web-based coupon book designed as a digital gift. This project replaces paper coupons with a modern UI, allowing the user to select and "redeem" rewards (like a dinner date or movie night) via interactive modal popups.

## üåü Features

* **Responsive Grid Layout:** Built with Bootstrap 5 to look perfect on mobile phones, tablets, and desktops.
* **Interactive Modals:** "Redeem" buttons trigger unique popups for each specific coupon.
* **Custom Design:** Features a "Cotton Candy" gradient theme (`linear-gradient`) and handwritten typography (Pacifico font).
* **Animations:** CSS hover effects on cards for a tactile feel.

## üõ†Ô∏è Technologies Used

* **HTML5:** Semantic structure.
* **CSS3:** Custom styling, gradients, and animations.
* **Bootstrap 5.3:** For the responsive grid system and modal components.
* **FontAwesome:** For vector icons (Utensils, Film, Hands, etc.).
* **Google Fonts:** Typography (*Pacifico* for headers, *Quicksand* for body text).

## üöÄ Installation & Getting Started

Since this is a static webpage, no complex installation (like Node.js or Python) is required.

### Prerequisites
You only need a modern web browser (Chrome, Firefox, Safari, or Edge).

### Steps to Run Locally
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/your-username/valentine-coupon-book.git](https://github.com/your-username/valentine-coupon-book.git)
    ```
2.  **Navigate to the project folder:**
    ```bash
    cd valentine-coupon-book
    ```
3.  **Open the file:**
    * Locate `index.html` in your file explorer.
    * Double-click it to open it in your default browser.

## üìñ Usage

1.  Send the link (or the laptop) to your significant other.
2.  They can browse the available coupons.
3.  When they click **"Redeem"**, a confirmation window appears.
4.  They can take a screenshot of the confirmation and send it to you to claim their reward!
